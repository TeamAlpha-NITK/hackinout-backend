# HackInOut backend

## Instructions

1. Install Scapy
```
pip3 install Scapy
```
 
2. Download pretrained statistical models for English
```
python3 -m spacy download en_core_web_md
```

3. Install djando
```
pip3 install Django
```

4. Install requirements.txt in `yolo` folder
```
pip3 install -r requirements.txt

```

## Usage
```
python3 manage.py runserver
```
